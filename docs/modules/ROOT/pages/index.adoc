= Quarkus logbook

include::./includes/attributes.adoc[]

A nice Zalando Logbook integration for Quarkus. Think the nice autoconfigured part of Zalando Logbook, but just for Quarkus.

== Installation

If you want to use this extension, you need to add the `io.quarkiverse.logbook:quarkus-logbook` extension first to your build file.

For instance, with Maven, add the following dependency to your POM file:

[source,xml,subs=attributes+]
----
<dependency>
    <groupId>io.quarkiverse.logbook</groupId>
    <artifactId>quarkus-logbook</artifactId>
    <version>{project-version}</version>
</dependency>
----
